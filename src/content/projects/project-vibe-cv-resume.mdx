# Vibe CV Resume

## Project Description

Vibe CV Resume is an open-source toolkit that enables AI coding agents (such as Claude Code, Cursor, and Windsurf) to maintain, update, and tailor your CV and resume using LaTeX and Git version control. Instead of manually editing documents, you describe the changes you want in natural language and let the AI agent handle the formatting and content updates.

## Project URL

[github.com/madnanrizqu/vibe-cv-resume](https://github.com/madnanrizqu/vibe-cv-resume)

## Key Features

- **AI-Agent Driven**: Use natural language instructions to update your CV — the AI agent handles LaTeX formatting
- **LaTeX-Based**: Professional, publication-quality typesetting with full control over layout
- **Git Version Control**: Track every change to your CV with full history and easy rollback
- **Docker Containerized**: No local LaTeX installation needed — compile PDFs in a reproducible Docker environment
- **Multiple Variants**: Easily maintain tailored versions of your CV for different roles or industries

## Technology

LaTeX, Docker, Git, AI Agents (Claude Code, Cursor, Windsurf)
